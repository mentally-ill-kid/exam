# Тестовое приложение для практики экзамена

Приложение было написано ради подготовки

## Как запустить

`git clone https://github.com/mentally-ill-kid/exam.git`

зайти в склонированную папку

`cd exam`

Создать виртуальное окружение

`python3 -m venv venv`

Активировать виртуальное окружение

`venv/bin/activate`

Скачать зависимости

`pip install -r requirements.txt`

Запустить приложение
`python3 main.py`
